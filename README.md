## Project Overview

AtliQ Hardware is growing rapidly in recent years and decided to
implement data analytics using Power BI for the first time to surpass
competitors and make data-driven decisions.

This project provides answers to stakeholders across Finance, Sales,
Marketing and Supply Chain functions through an end-to-end Business
Intelligence solution.

This project was developed by following the Codebasics Power BI Course.

## Company Background

AtliQ Hardware is a global company selling computers and computer
accessories through: - Retailers - Direct Channel - Distributors

The company experienced losses due to decisions based on intuition and
spreadsheets while competitors adopted analytics teams and data-driven
decision making.

## Questions Asked Before Building the Dashboard

-   What is the project deadline?
-   Are preview versions required?
-   What are stakeholder expectations and concerns?
-   Who will use the dashboard?
-   What can go wrong?
-   What data is required?
-   Are there design requirements from stakeholders?
